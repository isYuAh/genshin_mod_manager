## 准备工作
1. 首先，你需要安装一个Node.JS的环境，因为后端部分是基于node搭建的
2. 到`index.js`的目录下，执行  
```shell
npm install
```
## 使用方法
```shell
node ./index.js
```
## 说明
当你执行之后，这个api服务就会挂载，默认是在`:7777`端口上挂载
